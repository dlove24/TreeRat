Detailed Description

Defines an individual DNS record, used as the return type by many of the methods of the DNSList class.

For simple queries, especially those returning only a single record, it is easier to use the methods of the DNSQuery class directly. Only if the application requires more direct control of the data returned by a DNS query (e.g. custom handling of an MX record) should the methods of this class be called directly.

const boost::asio::ip::address DNSRecord::to_ip ( void  )
inline

Convert the internal DNS resource representation to a boost::ip::address.

Note
Only certain DNS names can be converted directly to an IP address (namely those from A and AAAA records). If the address cannot be converted this class will throw a DNSNameConversionException. If you don't want to handle exceptions, check the type before calling this method.
Return values
boost::asio::ip::addressAn IPv4 or IPv6 address record. We don't actually care which style of IP address we return: it is up to the caller to ensure they request the correct type.

Example Usage:

DNSRecord dns_name(a_dns_resource);
boost::asio::ip::address address{dns_name.to_ip()};
